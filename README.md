# LocalizedTest
学习本地化/国际化/多语言支持，实现App名称、内容文本在不同语言下显示不同语言的内容
## 介绍
本工程有三个功能：  
1. app名称根据语言环境显示对应语言  
2. 内容文本根据语言环境显示对应语言  
3. 获取当前使用的语言环境
